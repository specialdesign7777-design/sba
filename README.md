# Добавление тренера

Чтобы добавить нового тренера на сайт, выполните следующие шаги:

### 1. Загрузка фотографии тренера
- Загрузите фотографию тренера в папку `/public/coaches`.
- **Важно:** используйте латинские буквы в названии файла (например, `coach.jpg` или `ivanov.jpg`).
- Нажмите **Commit changes** для сохранения файла.

### 2. Редактирование файла с данными
- Перейдите в `/src/data` и откройте файл `about_coaches.json`.
- Нажмите на иконку редактирования (карандаш).

### 3. Добавление информации о тренере
- Внесите данные нового тренера, соблюдая структуру существующих записей.
- В поле `"img"` укажите **точное название файла** из пункта 1 (например, `"coach.jpg"`).

### 4. Сохранение изменений
- Нажмите на зелёную кнопку **Commit changes**.
- Подтвердите сохранение, нажав **Commit changes** ещё раз в диалоговом окне.

### 5. Ожидание обновления
- Изменения отобразятся на сайте примерно через **1-2 минуты** после выполнения всех шагов.
